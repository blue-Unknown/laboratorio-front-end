O seria o elemento ?

Esse elemento é bem usado para certas ocasiões, como deixar um texto não tão importante menos visível, ou colocar uma informaçãozinha pequena embaixo de uma imagem ou quanquer coisa que tiver muita informação, tipo um complemento.

Vamos ver um exemplo:

Olha como está esse texto, insira muchas informações aqui

Esse textinho ta bem pequeno, insira alguma informação complementar ou não muito importante aqui
